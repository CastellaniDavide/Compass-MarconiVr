## Description
Compass is an app (at the moment only for Android) for orientation in the school Marconi. 

---
## Who do this? 
The work was made by Marconi's students:
- Castellani Davide (contacts@castellanidavide.it)
    - group leader
    - app maker (with Android Studio)
    - ...

---
## Final Objectives
- light app
- offline mode
- upgrade notification
- sync between devices (when it's online)
    
---
## What I used?
I used:
- Android Studio
- Bootstrap Studio
- Imagination

What I need in the future:
- a database
- a storage w/ web access
